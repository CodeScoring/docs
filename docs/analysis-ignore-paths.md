# Пути анализа и исключения

В анализе сконфигурированы исключения для путей, по которым **не производится** поиск манифестов, файлов и не происходит анализ качества (формат выражений — glob):

- `**/.git*`
- `**/.git/**`
- `**/fixtures/**`
- `**/tests/**`
- `**/doc/**`
- `**/docs/**`
- `**/samples/**`
