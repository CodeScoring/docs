    CodeScoring.TQI¶

    Общее описание¶

    CodeScoring.TQI – это модуль для анализа качества собственного исходного кода организации, определяющий основные параметры технического долга.

    Основные функциональные возможности модуля позволяют:

    • Просматривать динамику проекта с подсчетом цикломатической сложности кода;
    • Строить профили авторов с сравнением схожести;
    • Отслеживать дубликаты кода внутри и между проектов.
