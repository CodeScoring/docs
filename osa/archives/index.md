# Сканирование архивов

Плагины CodeScoring.OSA поддерживают сканирование архивов в следующих форматах:

| Экосистема | Формат архива                                                 |
| ---------- | ------------------------------------------------------------- |
| Maven      | `.jar`, `.war`, `.ear`                                        |
| NPM        | `.tgz`                                                        |
| PyPI       | `.zip`, `.tar`, `.tgz`, `.tar,gz`, `.tar.bz2`, `.egg`, `.whl` |
| Nuget      | `.nupkg`                                                      |
| Cocoapods  | `.tar.gz`, `.zip`                                             |
| Go         | `.mod`, `.zip`                                                |
| Gems       | `.rz`, `gz` `.gem`                                            |
| Debian     | `.deb`, `.xz`, `.gz`                                          |
| Yum        | `.rpm`                                                        |
| Alpine     | `.apk`                                                        |
| Docker     | `.json`                                                       |
